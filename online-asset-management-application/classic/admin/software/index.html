  • Software


      By default, the software feature is disabled. The administrator can enable it from the general settings. Once enabled, two software tabs will be displayed on the home page, as shown below.


      The first tab is used to synchronize the O365 license. This synchronization relies on the API configuration in the general settings. Please refer to the link below for API configuration details. API Configuration

      Click on the sync icon to synchronize the O365 license, as shown in the image below.



      To view the users to whom licenses have been assigned, click on the item link. The License Details panel will appear, and from there, click on the "Full Sync" tab, as illustrated below.



      Note: Assigning or returning O365 licenses is not supported within the Asset Management module.

      The second Software tab allows you to add software records, including quantities. Once added, you can assign the software to users as needed.



      Assigning and Returning Software
      In the Action column, use the respective icons to assign or return software. To modify software details, click the Edit (pencil) icon. To remove a software entry, click the Delete icon.

  • Bulk Software


      By default, the software feature is disabled, the admin can enable it from general settings. Once it is enabled, two software tabs will display on the home page as shown below.


      You can import up to 1000 software records at once using the Import Software option. Download the sample file and populate it with data in the prescribed format.


    • Follow the below instructions before adding bulk assets:

    1. Please download the sample file above, fill in the software details as per the instructions provided, save the file, and upload it.
    2. Ensure that separators and currency symbols are removed from the cost/value fields.
    3. Remove any commas from the notes or other fields to avoid conflicts with CSV formatting.
    4. The date format should be MM/DD/YY. This format will be applied during the upload process, but you can adjust it in the settings if needed.
    5. Mandatory fields such as Software Name, Quantity, Price per Unit, License Type, Recurring, Renewal, Renewal Frequency, and Status must not be left empty.
    6. Do not change the order of the columns or add new ones, as this may result in incorrect data mapping.
    7. You can upload a maximum of 1,000 software entries at once using the "Import Software" feature.

    Choose the data loaded CSV file from the system and upload the file.
