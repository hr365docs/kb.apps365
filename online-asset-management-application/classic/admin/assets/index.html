  • Admin Guide

  • Add Assets

      You can add assets from this section. On the Add Assets page, you can enable the fields you need and disable those that are not required.

      However, mandatory fields such as Asset Name, Asset Type, and Category cannot be disabled.

      To configure the fields, click on the Asset tab to open the Add Asset page. Then, click on the gear icon at the top-right corner to open the column display settings.


      Use the checkboxes to enable or disable specific fields, and click the Save button to apply the changes.


      Additionally, you can set default values for Category, Subcategory, Asset Type, and Status from the side navigation panel under the Administration section.

      You can add assets from two places:

      1. Asset tab
      2. Assets from side navigation panel under Administration

      Asset Tab

      To add an asset type, follow the steps below:

      1. Click on the +Asset tab.

      2. Fill in the mandatory fields such as asset name, category, and asset type. Other fields are optional. You can also upload an image or add a document. Enable the toggle switch to set Asset Expiry Notifications for assets.

      3. Finally, click on the Save button or save icon.

      4. The created asset will be displayed on the Home Page and the assigned asset table.

      To generate a system barcode, please enable the “System Generated Barcode” option in the general settings.

      Adding an Asset as Multiple: Click on the Add Multiple button, fill in the mandatory fields, enter the number of assets you want to create, and then click on the Submit button as shown below.

      Do not enter serial numbers when creating multiple assets.

  • Import asset or Bulk Assets

      You can add up to 1,000 assets at once using a CSV file. Download the sample file and enter the data in the specified format.



      Follow the below instructions before adding bulk assets:

      1. Add the Asset ID in the specified column; this field is mandatory.
      2. Remove any separators from the cost or value fields.
      3. Use the date format MM/DD/YYYY when uploading data. The display format will follow your settings.
      4. Do not change the order of the columns or add new columns, as this may result in values being placed in the wrong columns.
      5. Mandatory fields such as Asset Name, Asset ID, Asset Type, and Category must not be empty.
      6. The barcode should not exceed 8 digits.

      To add bulk consumables, follow the steps below:

      1. Click on the +Bulk Assets or +Import Assets tab. This will display the Add/Update Multiple Assets option.
      2. Select the CSV file with the data from your system and upload it.
      3. Finally, click the Save button.



      From here you can update the bulk asset details.

  • Assign Assets

      From this section, you will be able to assign assets. To add an asset type, follow the steps below:

      1. Click on Assign Assets in the top navigation panel.


      2. Click on the link under the Action column. This will display a pop-up.


      3. Enter the User name, select the Project, and specify the Return Date.

        You can assign assets using User / Group or Location.


      4. Finally, click on the Assign button.


      5. The assigned asset will be displayed on the Return Assets page.

  • Return Assets

      From this section, you will be able to return assets. To add the asset type, follow the steps below:

      1. Click on Return Assets in the top navigation panel.


      2. Click on the link under the Action column. This will display a pop-up.


      3. Finally, click the "Yes, Return It" button to complete the process.


      4. The returned asset will be displayed on the Assign Assets page.

  • Booking Details

      Cancelling Booked Assets and Updating Booked Assets

      In case you need to cancel a booked asset or edit the booking details, you can do so from the Booking Details page. To cancel a booked asset, simply click on the Cancel link in the Action column. To update a booked asset (such as changing the assign or return date), click on Update in the Action column. A pop-up will appear where you can modify the date, and then click on the Yes, Book It button to confirm the changes. All booked assets can be found on the Booking Details page.

  • Approve/Reject Asset

      By default, Approve/Reject Asset feature is not enabled, the admin can enable it from Users from the side panel under settings.


      From this section, you can able to Approve/Reject assets. Select the checkbox under Action for booked assets and select Approve or Reject button.
