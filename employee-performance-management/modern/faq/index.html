    How to Download and Share a HAR File for Support?

    When you experience issues such as being unable to add users for the KPI cycle or appraisal cycle, or if entered data disappears after refreshing the page, the support team may request a HAR file. A HAR (HTTP Archive) file captures detailed logs of your browser’s interactions with the system. This helps us analyse what happened on your screen and identify the root cause quickly and accurately.

    Why the Support Team Needs a HAR File

    A HAR file allows us to:

    1. Review each action taken in your browser.
    2. Detect any network-related issues, failures, or delays.
    3. Identify technical errors that may not be visible on the screen
    How to Download a HAR File

    The steps to open Developer Tools and save a HAR file differ based on your operating system.

    For Windows Users:
    1. Open Google Chrome or Microsoft Edge.
    2. Navigate to the PM365 page where the issue occurs.
    3. Open Developer Tools using any one of the following:
      * Press F12.
      * Press Ctrl + Shift + I
      * Click the three dots menu (⋮) → More Tools → Developer Tools.
    4. Click on the Network tab inside the Developer Tools panel.
    5. Ensure that you clear the logs in network tab (as shown in the clips).
    6. Reproduce the same scenario where you are encountering the issue.
    7. Click on download and download the HAR file.

    For Mac Users:
    1. Open Google Chrome or Microsoft Edge.
    2. Navigate to the PM365 page where the issue occurs.
    3. Open Developer Tools using one of the following methods:
      • Press Command + Option + I
      • Click the three dots menu (⋮) → More Tools → Developer Tools
    4. Click on the Network tab inside the Developer Tools panel.
    5. Ensure that you clear the logs in the Network tab (as shown in the clips).
    6. Reproduce the same scenario where you are encountering the issue.

    How to Create & Assign a Template?

    Step 1 – Navigate to the Administration Menu

    From the Home page, click the Administration button on the left navigation bar.


    Step 2 – Open the Templates Section

    From the left navigation bar, a dropdown menu with several configuration options will visible. In that dropdown scroll down and click on Template(s). This opens the Templates panel, which contains three tabs: Global Template(s), My Template(s), and Assign Template(s).


    Global Template(s) contains system-wide templates visible to all users. My Template(s) shows templates you have created. Use the Assign Template(s) tab to assign templates to specific employees or groups.


    Step 3 – Create a New Template

    To create a new template, click the + Add Template button in the Global Template(s) or My Template(s) tab. The Add Template form will open on the right side of the screen.


    Fill in the Template Details

    Complete the following fields to define the template:

    1. Name – Enter a clear, descriptive name for the template (e.g., "Product Manager Template"). This name will appear in the template list and during assignment.
    2. Clone Template – Optionally select an existing template to clone. This copies all questions and settings from that template into the new one, which you can then modify as needed.
    3. Description(Optional) – Provide a brief description of the template's purpose — for example, the type of employee it is intended for or the review focus area.
    4. Job Title(s) (Optional) – Select one or more job titles this template applies to. If an employee's job title matches, the template can be auto-assigned without any additional manual steps.
    5. Company (Optional) – Select the company or legal entity this template is scoped to.
    6. Department (Optional) – Select the department the template applies to.
    7. Location (Optional) – Select the office location the template is applicable for

    Auto-Assignment: If an employee satisfies all the specified conditions (job title, company, department, and location), the template will be automatically assigned to them — skipping the need for manual assignment.

    Step 4 – Verify Total Weightage

    Before saving, ensure that the combined weightage across all KRA(s), Goals, and Competency & Skills adds up to exactly 100.


    Tip: Distribute the weightage thoughtfully across sections to reflect your organisation's performance priorities. For example, KRAs might carry 60%, Goals 25%, and Competency & Skills 15%.

    Step 5 – Verify Total Weightage Module wise

    Before saving, ensure that the weightage within each module KRA(s), Goals, and Competency & Skills adds up to exactly 100%. Each module is calculated independently for example, if you add 4 KRAs, their combined weightage must equal 100%; the same applies to Goals and Competency & Skills


    Step 6 – Save the Template

    Once all KRAs, Goals, and Competency & Skills have been added and the total weightage is 100, click the Save button at the bottom of the Add Template panel. The template will be created and saved to the system.


    Step 7 – View Your Saved Template

    After saving, navigate to the My Template(s) tab in the Templates panel. Your newly created template will appear in the list with its name and description.


    Step 8 – Assign the Template

    To manually assign a template to specific employees or groups, navigate to the Assign Template(s) tab. Here you can assign templates based on a variety of criteria:

    • Appraisal Cycle – Assign to all employees in a specific cycle.
    • Department – Assign to all employees in a specific department and so on.

    Select the template from the Select template dropdown, choose the assignment method from the Assign Selected Template by dropdown, and fill in the Assigned To field. Then click Save to complete the assignment.


    How to create an appraisal cycle and add the users in appraisal cycle

    This guide walks you through setting up and configuring an Appraisal Cycle in the system, from navigating to the settings page all the way through saving the completed cycle.

    Overview

    An Appraisal Cycle defines the time and structure used to evaluate employee performance. It controls key parameters such as review duration, participants, and stage timelines. Once configured and saved, the cycle becomes available for employees and appraisers.

    Step 1 – Access the Settings Panel

    Locate the Gear icon in the left-hand navigation bar. This icon provides access to all administrative and configuration settings for the application.


    Step 2 – Open the Appraisal Cycle Section

    Within the Settings panel, navigate to the Features column on the right-hand side. Click on Appraisal Cycle to open the management page.


    Step 3 – Configure the Appraisal Cycle

    a. Create a New Cycle

    Click the + Add button at the top-left of the Appraisal Cycle page. The "Add Appraisal Cycle" panel will open.

    b. Name of the Appraisal Cycle

    Enter a descriptive name such as "Annual Review 2025" or "Q3 Performance Cycle".

    c. Select Appraisal Cycle Duration

    • Quarterly
    • Half-Yearly
    • Yearly

    d. Appraisal Cycle Start Date

    Set the beginning date of the evaluation period.

    e. Appraisal Cycle End Date

    Set the ending date of the evaluation period.

    f. Review Start Date

    This is when the review workflow becomes active for employees and appraisers.

    g. Number of Days for Self Review

    Define how many days employees have to complete their self-assessment.

    Note: If not completed before deadline, employees must request HR to reopen.

    h. Number of Days for Appraiser's Review

    Define the number of days for managers to review employee submissions.

    i. Number of Days for Collaboration

    Set the days for HR and leadership calibration to ensure fairness.

    j. Number of Days for Review Conclusion

    Define the days to finalize ratings and prepare communication to employees.


    Step 4 – Save the Appraisal Cycle

    Click the Save button at the bottom of the panel. The new cycle will appear in the Appraisal Cycle list.


    Adding Employees to the Appraisal Cycle

    After saving, a new window opens where you can add employees. Initially, the list will show “No data available.”

    Step 5 – Manage Appraisees Panel

    Click the + Add button to open the Employee Selection dialog.


    Step 6 – Select Employees Using Checkboxes

    Check the box next to each employee you want to enroll. You can select multiple employees.

    Click Save to add them to the cycle. Click Cancel to close without saving.


    Once you have selected all the employees you want to include, click the Save button at the bottom of the Employee Selection dialog. The selected employees will be added to the appraisal cycle and will appear in the "Employees in Appraisal Cycle" panel.
    If you wish to cancel without making any changes, click the Cancel button to close the dialog without saving.


    Tip: You can return anytime to add or review enrolled employees, even after the cycle goes live.
