  • Consumable

      Consumables (also referred to as consumable goods, non-durable goods, or soft goods) are items intended for consumption. These are products that consumers use regularly, meaning they "get used up" or are discarded after use.

      By default, the consumable feature is disabled. However, the administrator can enable it via General Settings. Once enabled, four consumable tabs will appear on the homepage, as shown below:


      If you wish to rename the module title, this can be done in General Settings.

      The first tab is designed for adding consumable items in bulk. For further details, refer to the Bulk Consumable section.

      The second tab allows for adding individual consumable items. For more information, refer to the Add Consumable section.

      The third tab is used for managing consumable items, including placing orders and editing or deleting items in bulk. For further guidance, refer to the Consumable Item section.

      The fourth tab enables users to request consumable item orders. For more details, refer to the Request Consumable Item section.

  • Add Consumable Item

      To add a consumable item, follow the steps below:

      1. Click on the +Consumable Item button, which will open the Consumable page.



      2. Fill in all the mandatory fields, including Product ID, SKU ID, Item Name, Status, Item Description, Category, Sub Category, Quantity, and Total Price. You can add multiple rows for categories and delete a row by clicking the delete icon.

      3. Once all fields are filled, click the Save button to save the consumable item.

      4. The newly added consumable item will be displayed on the Consumable Item page.


      5. Click the Export Barcode button to generate and print the barcode, as shown in the image below.



      6. After clicking on the Export button, You can generate the barcode from here and print the barcodes as shown in the below image.

  • Bulk Consumable

      You can add 1000 consumables in one go using CSV file. Download the sample file and add data in the specified format.


      Follow the below instructions before adding bulk consumable items:

      1. Enter the Product ID in the specified column. This field is mandatory.
      2. Remove any separators from the cost or value fields.
      3. Ensure the date format is MM/DD/YYYY for uploading the data. The display format will follow your settings choice.
      4. Do not alter the order of the columns or add new columns, as this may result in data being entered into the wrong fields.
      5. Mandatory fields such as Asset Name, Asset ID, Asset Type, and Category should not be left empty.
      6. Do not change the order of the columns or add new columns, as this may lead to incorrect data mapping.
      7. The Barcode should not exceed 8 digits in length.

      To add bulk consumable items, follow the steps below:

      1. Click on the +Bulk Consumable tab, which will open the Add Multiple Consumable Items page.
      2. Select the CSV file with the loaded data from your system and upload the file.
      3. Finally, click the Save button to complete the process.

  • Consumable Item

      From this section, you can edit, delete, and place orders for consumable items.


      To place an order for a consumable item, follow the steps below:

      1. Click on the SKU ID link, which will display a popup.


      2. Select the appropriate category, subcategory, and shipping method (pickup/ship).


      3. Finally, click the Place Order button.


      4. The placed consumable order will be displayed on the Approve/Reject Consumable page.


      Edit: You can edit the details of a consumable item by clicking the edit icon under Action. Update the fields and click the Update button.


      Delete: You can delete a consumable item by clicking the delete icon under Action.

  • Request Consumable

      From this section, you can initiate an order for the consumable item.


      To request consumables, follow the steps below:

      1. Click on Request under Action to display the "Place Order" popup.


      2. Select the appropriate category, subcategory, and choose between pickup or shipping options.

      3. Finally, click on the Place Order button.


      4. The requested consumable item will appear on the Approve/Reject Consumable page.

  • Approve/Reject Consumable

      By default, the Approve/Reject Asset feature is disabled. The administrator can enable this feature for Users via the settings in the side panel.


      From this section, you can approve or reject assets. Select the checkbox under Action for the booked asset, then choose the Approve or Reject button accordingly.
