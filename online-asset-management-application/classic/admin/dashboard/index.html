  • Dashboard


      The dashboard consists of four rows containing various graphs, tables, and other elements. You can choose to hide or display these rows through the dashboard settings, which can be accessed from the settings menu in the side panel.

      Row 1


      This section displays various asset statuses along with the corresponding numbers, indicating the count of assets based on their status, as shown below. You can add additional statuses according to your requirements by navigating to the "Status" option under the administration section in the side panel.

      Row 2


      In this row, you will find graphs displaying total assets, assets by department, and assets by location.

      Total Assets Graph: This graph shows the number of assets categorized by their type and subcategory. Hover your mouse over any part of the graph to view the asset count for that category.

      Assets by Department: This graph displays the number of assets assigned to each department.

      Assets by Location: This graph shows the number of assets assigned to various locations.

      Row 3


      Assets Availability: This table displays the total number and the available quantity of assets, categorized by subcategory.

      Expiring Soon: The calendar shows assets that are nearing their expiry date, with each asset displayed according to its expiration date.

      Assets by Vendor: This graph provides details on the number of assets, categorized by their respective vendors.

      Row 4


      Assets by Status: This graph displays the number of assets categorized by their current status.

      Recent Activity: This table shows your recent activity for assigned and returned assets. The most recently assigned asset will appear at the top of the table, while the most recently returned asset will also be displayed at the top. You can filter and view specific entries using the dropdown menu.
