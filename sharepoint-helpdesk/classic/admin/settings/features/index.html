    Features

    From this section you can customize the features of helpdesk application.

    Canned Responses

    Canned response settings are accessible to admins, supervisors, and agents from the settings page. These pre-defined responses can be used when commenting on a ticket. To use a canned response, click the reply option, select a response from the canned response panel, and click the "Add" button to insert it into the comment.

    There are three types of canned responses available:

    Global Canned Response: These are public responses that can be created and used by all team members.

    My Canned Response: These are user-specific responses, meaning only the creator can view and use them when commenting on a ticket.

    My Signature: Team members can set their own default signature, which will automatically appear when adding comments on a ticket. These signatures are individual-specific.



    To create Canned Responses, follow the steps below:

    This applies to global canned responses, my canned responses, and my signature.

    1. Navigate to the Settings menu and click on Canned Responses under Features.

    2. Click on +New to open the New Canned Response form.


    3. Enter the Canned Response Title and the Canned Message.

    4. Click the Save button to save the new canned response.


    Delete: Click the delete icon, this displays a dialog box. Click Yes button.

    Edit: You can edit Title and Message Body. Click the edit icon, this displays Edit Canned Response. Update the Title and/or Message Body and click on Update button.

    Customer Satisfaction

    A survey link notification is sent to the ticket requester once their ticket has been resolved or closed. From here survey link notification can be set either for resolved or closed tickets or sent manually.

    A survey link notification is automatically sent to the ticket requester once their ticket is resolved or closed. From this section, you can configure when the survey link notification should be sent, either upon ticket resolution, closure, or manually.


    If Allow agent to send a survey link is selected, a Click here link will appear when the ticket status is set to "Resolved" in the properties panel on the comment page. The ticket assignee can then click the link to send the survey notification to the requester.



Delete: Click the delete icon, this displays a dialog box. Click Yes button.

Edit: You can edit Survey Name, Point scale, Thank you page and survey link in emails.

    Custom Column

    From this section, you can add, edit, and delete custom columns.

    To add a new column, follow the steps below:

    1. Click on + New Column, which will open the New Column page.


    2. Select the checkbox from the category options.

      You have two category options to choose from: Ticket Description and Ticket Property. - If you select Ticket Description, the details will be visible on the comment page. - If you select Ticket Property, the details will be visible on the home page.

    3. Enter the column name, select the column type, and mark it as active. Then, click the Save button.


    You can create sub column for main coulmn of only choice type.

    To add a new sub-column, please follow the steps below:


    1. Click on + New Sub Column, which will open the New Sub Column page.


    2. Select the appropriate Column, Sub Column Type, Column Values, Sub Column Values, and set the Active status. Additionally, provide a name for the Sub Column.


    3. Click the Save button to finalize the creation of the sub-column.

    Business Hours

    From this section, the admin has the ability to add, edit, and delete business hours.

    To add business hours, please follow the steps outlined below:


    1. Enter the Name and Description of the business hours.

    2. Select the Helpdesk Hours for the specified business hours.

    3. Click the Submit button to save the business hours.

    Column Settings

    From the Column Settings section, the admin can manage the visibility, sorting, and display order of columns.


    1. The Column Settings panel consists of six tabs: Unassigned, Team Tickets, Team Closed, Team Resolved, My Tickets, and Report.


    2. Using this feature, you can easily rearrange the columns by dragging them to your desired position.



    3. Select the appropriate tab where you wish the columns to appear. Then, check the boxes next to the columns you want to display. Finally, click the Save button.


    4. The selected columns under the Team Tickets tab will appear on the Home Page.

    Define SLA Policies

    An SLA (Service Level Agreement) policy allows you to define performance standards and set target deadlines for ticket resolution. These policies help ensure that your support team resolves tickets within a specified timeframe. If any SLA policy is breached, an escalation notification will be sent to the relevant team.

    By default, SLA policies are disabled. You can activate them by toggling the Enable SLA switch.


    You can select the priority level from the priority dropdown.


    SLA Calculation: Two types are available: Initial Response SLA and Resolve Time SLA. Select the desired option from the checkboxes and click the Save button to apply the settings.


    Apply SLA Policies

    The defined SLA policies can be applied from this section.

    Follow these steps to apply SLA policies:

    1. Click the Configure button next to Apply SLA Policies to begin.


    2. The Apply SLA Policies panel will open. There are four dropdown menus available: Teams, Services, Request Type, and SLA Policy.


    3. After making your selections from the dropdown menus, click the Save button to apply the SLA policies.


    Time Tracking

    Admin, supervisor, and agent can track time on their tickets. They can add time during the comment of the tickets.

    By default time tracking are disabled, you can enable it by clicking Enable Time Tracking toggle switch. You can enable Make Time Tracking Mandatory also.



    To use the time tracking, follow the below steps:

    1. Go to the home page and click on the Ticket ID to open the comment page.

    2. On the comment page, click the Reply button. This will display a pop-up where you can write your comments.


    3. Click the Update button, which will open the Track Time pop-up.

    4. Enter the time taken to resolve the ticket in minutes.


    You can edit the time spent by clicking on the edit icon.


    You can find the Track Time column on Home Page.
