    Home Page

    On the Home page, you can view data for the following sections: Unassigned, Team Tickets, Team Closed, Ticket Mailbox, My Tickets, My Requests, and My Closed Tickets.

    There are Three options in Home Page to display the tickets

    1. Classic View
    2. Kanban View
    3. Modern View


    Classic view

    • Tickets will be highlighted with the different colours on the basis of ticket Priority
    • Bell icon notification for unread tickets on the home page.


    Kanban view

    You can drag and drop tickets to change the status of ticket in kanban view


Modern View



Sort

Click this button to sort data by various fields, such as Teams, Services, Request Type, and more, in ascending or descending order.



Sorting options include Teams, Services, Sub Services, Request Type, Priority, and more, with controls for both ascending and descending order.



Ticket Creation

Users can raise tickets for any service. To create a ticket, follow these steps:

  • Click on the + New Ticket button on the Home page, which will display the new ticket page.


To create a new ticket, fill in the relevant fields such as Title, Teams, Services, Priority Type, and Request Type. Add a description of the issue, CC users if needed, and choose the communication method before submitting the ticket.


  • When the toggle is turned on, the ticket form will appear as shown below.


  • When the toggle is turned off, selecting a team in the new ticket panel is not required. The team will be automatically assigned based on the selected service and subservice.

Custom Views

Open the select box to add a custom view.


Select the Configure View option.


Now, click the Add button.


The admin can create different views of the tabs and tables of the home page.


When a user configures a view, they have the option to share it within the same site. The shared view will be visible only to the selected user. To share a view, click the Share icon and choose the user you want to share it with.



Assign tickets and update status from the home page.

To update the status of a ticket, click on the Status dropdown and select the appropriate option from the list.


Fill the required fields. then click on submit button


Bulk Assign ticket

In the 'Unassigned' tab, users can efficiently manage tickets by selecting multiple unassigned tickets for bulk updates. This functionality allows users to assign a team, designate a specific user, update the ticket status (e.g., 'Resolved'), and apply all changes with a single click on the 'Update' button.


Users can now efficiently update the status, team, and assignee for multiple unassigned tickets at once using the multi-select option.
