    Reports

      We have four options within the report: All Reports, Project Reports, My Teams, and Invoice Report.


      The Daily Report: Offers a detailed breakdown of data for a single day. This report is ideal for monitoring daily activities, performance, and addressing issues that may arise.


      The daily report displays all date columns and shows records for the month, filtered by user, client, department, status, project, and task.


      A "User Status" column has been added to the report pages, displaying whether a user is active or inactive. The status information is directly fetched from the user page.


      The Weekly Report: Provides a summary of data and key metrics for the week. It highlights trends, changes, and performance indicators on a weekly basis, enabling users to track progress and make comparisons between weeks.


      The weekly report displays all date columns and shows records for the week, filtered by week, client, project, task, user, project manager, department, and status.


      The Monthly Report: Provides a summary of data and performance metrics over the course of a month. This report is useful for identifying long-term trends and understanding broader performance changes.


      The monthly report displays all date columns and shows records for the month, filtered by month, client, project, task, user, project manager, department, status, and assigned to.


      The Full Report: A comprehensive view of data across all time frames (daily, weekly, monthly). It allows users to analyze a more complete dataset for in-depth analysis and decision-making.


      The full report displays all date columns and shows records for the selected date range, filtered by date range, user, client, project, task, assigned to, department, and status.


      Project Reports: The Project Report Page provides a centralized view of all ongoing and completed projects. Users can view a list of all projects, and by clicking on a specific project, they can access a detailed summary of that project, including key information and relevant data.


      My Teams: The Team Page provides an overview of the team’s activities, enabling users to track and monitor team performance and engagement. It offers a snapshot of the tasks, goals, and progress of each team member or the team as a whole.


      The Project section provides a comprehensive overview of all ongoing and completed projects, including key details such as project status, timelines, and objectives. This helps stakeholders monitor progress and make informed decisions based on project performance.


      The Team Members section lists all individuals involved in various projects, detailing their roles and responsibilities within the team. This facilitates better collaboration and accountability by showcasing each member's contributions and the time dedicated to project tasks.


      The Invoice Module: Provides a comprehensive view of all invoices within the application. It allows users to view, manage, and track invoices efficiently, helping streamline billing and payment processes.
