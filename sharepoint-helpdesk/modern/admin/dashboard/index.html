    Dashboard

    The dashboard provides a comprehensive overview of ticket data categorized by Status, Departments, Priority Type, Request Type, and Survey.

    Line Chart 1: Displays the number of open tickets for the current month. Users can filter this data based on specific teams to analyze team-wise performance.

    Line Chart 2: Illustrates the average SLA response time within a selected time frame or specific date, helping monitor service responsiveness.

    Bar Chart: Represents the weekly average SLA response time for the selected period, enabling trend analysis over time.

    Donut Charts: Provide a visual breakdown of ticket volume based on Request Type, Departments, and Priority Type, offering quick insights into ticket distribution.


    User can also expand the chart with the help this expand icon




    User can filter the Dashboard data globally with the help of this globally filter icon



    Select the dropdown option and click on Apply Filter button.



    Customer Satisfaction

    You can view customer satisfaction data represented as a percentage for the selected date range. The dashboard allows filtering by individual teams, with the data available in both chart and table formats for detailed analysis.



    Dashboard charts can now be filtered and viewed based on different ticket statuses, enabling more focused and meaningful insights.



    The dashboard's visual presentation has been enhanced to effectively accommodate up to 50 services, ensuring a clean layout, improved readability, and a seamless user experience. Services are now displayed in a scrollable format with the total number of services clearly indicated.


    The icon highlighted in red represents the "View Options" feature. Clicking this icon opens a dropdown menu with two options: Graphical View and Tabular View.


    Selecting Graphical View displays the data in a chart format, allowing users to quickly analyze and interpret information through intuitive visual representation.


    Selecting Tabular View presents the data in a table format, offering a structured and clear view for users to efficiently understand and compare the data.
