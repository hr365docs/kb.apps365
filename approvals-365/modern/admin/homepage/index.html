  • Home Page

    • 1. Approvals 365 Dashboard

    The dashboard is the main landing page of Approvals 365. It displays two primary tabs:

    • Received: Shows approval requests assigned to you.
    • Requested: Displays approval requests you have submitted.

    From here, users can create new requests or manage templates.


    2. Template


    The Template allows administrators to select predefined templates or create new templates from scratch.

    Admins can also click Create from Scratch to design a custom approval process.


    3. Template Basic Settings

    In the Basic Settings section, administrators define core template information such as:

    • Template Name
    • Category
    • Description
    • Default Template Toggle

    This step ensures the template is properly categorized and easily identifiable.


    4. Form Design Interface

    The Form Design section allows administrators to customize the request form using a drag-and-drop interface.

    Admins can:

    • Add new sections
    • Drag fields from the right panel
    • Reorder fields
    • Delete unwanted fields

    5. Approver Configuration

    In the Approvers Settings step, administrators configure the workflow levels.

    Features include:

    • Assigning one or multiple approvers
    • Creating multi-level approval chains
    • Adding additional levels if required

    The workflow follows this structure:

    Request → Level 1 Approver → Level 2 (optional) → Complete


    6. Submitting a New Approval Request

    Users can submit a new approval request by selecting a template and filling out the required fields.

    Once submitted, the request is automatically routed to the configured approver.



    7. Approval Status & Tracking

    Each approval request includes a visual timeline that displays the workflow progress.

    Status indicators include:

    • Started
    • Notification Sent
    • Approver Assigned
    • Approval Pending
    • Approved / Rejected

    This provides full transparency and real-time visibility into the approval process.





    Approvals 365 helps organizations digitize manual approval workflows, reduce processing time, improve transparency, and ensure structured governance across departments.
