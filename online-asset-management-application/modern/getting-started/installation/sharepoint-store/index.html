Installation of Asset Management 365 from SharePoint Store

Prerequisites:

  1. The installation of the application is restricted to Global Admin or SharePoint Admin, who can complete the process by adhering to the steps provided below.
  2. The functionality of the application depends on APIs, which can be enabled by a Global Admin possessing the required permissions.

  1. Navigate to the App Catalog page using the following URL format:

    https://yourtenantname.sharepoint.com/sites/appcatalog/_layouts/15/tenantAppCatalog.aspx/manageApps

    Replace yourtenantname.sharepoint.com with your actual tenant name. For example: cubiclogics.sharepoint.com.

    Once on the page, click on the Classic Experience option.


  2. Click on the Settings gear icon, then select Add an app.




  3. Click on the SharePoint Store and enter Apps365 in the search box. All applications published by Apps365 will be displayed.



  4. Click on Asset Management 365 application.


  5. Click on Add to Apps site.


  6. Select the checkbox labeled "Enable this app and add it to all sites", and click on Add button.

    7. Once the application is installed, you can make the solution or application package available across all sites in the organization. If needed, you can later remove the app from all sites.

    By performing these actions, the selected application package will be downloaded from the Microsoft SharePoint Store into your tenant.


  7. Click on OK button.


  8. In the SharePoint Admin Center, from the left panel, navigate to the Advanced section and select API Access. Under the Pending Requests tab, you will find several requests, including Microsoft Graph, Presence.Read.All, User.Read.All, MailboxSettings.Read, Calendars.Read, Group.Read.All, Chat.ReadWrite, Chat.Create, and Mail.Read.Shared. For each request, select it and click Approve individually.



  9. The user can select and approve each permission individually before proceeding further.




  10. Navigate to the desired location (i.e., site collection or subsite) where you wish to install the Employee Directory 365 application.

    11. We have to add Asset Management 365 as the web part on the page.


  11. Click on New and select Page.


  12. Click on the + icon, then select Asset Management 365, as shown below.



  13. Finally click on Publish button.


  14. Finally, refresh the page.

There are two distinct methods of installation:

1. Package Upload Installation via the app catalog site.
2. Installation from the SharePoint store (AppSource).

Please note that deleting the package installation and subsequently reinstalling the app from either the SharePoint Store or the app catalog will not result in any data loss. This is because the application's data is securely stored in SharePoint lists, which remain intact regardless of the installation method used.
