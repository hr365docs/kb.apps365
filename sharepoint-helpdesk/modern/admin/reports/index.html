    Reports

      Admins and Supervisors have the ability to view and generate reports. To begin, filter the ticket data using your desired parameters such as date range, assigned to, teams (department or group), status, priority, request type, and services. Once filtered, the report can be downloaded using the Export to Excel option.


      When a user applies a global filter like "Current Month," "Last Month," or other predefined date ranges, the system restricts filtering strictly to that timeframe. As a result, other filters such as agent, team, priority, and status are disabled to prevent conflicting filters. To use other filters, the global filter must first be cleared or reset.


      Teams Reports


      Agent Report

      Admins and supervisors can access reports showing the number of tickets assigned to agents, categorized by ticket status and tracked time.

      Admins can view ticket data for all agents across all teams, whereas supervisors have access only to data pertaining to their respective teams.

      To generate an agent report, first select the desired date range, then choose the relevant teams from the filter dropdown. Click on the filter icon to apply.

      Use the Export to Excel button to download the filtered report as an Excel file.


      CSAT Report

      The Customer Satisfaction (CSAT) report presents individual survey responses submitted by users in response to feedback questions. CSAT measures how satisfied customers are with a product, service, or experience by asking them to rate their satisfaction level on a predefined scale.



      Display the total count of tickets in the report based on the applied filters or search criteria, enhancing visibility and tracking.


      When the user applies a filter, such as selecting "High" in the Priority filter, only tickets with "High" priority will be displayed. The total count of these filtered tickets will also be shown for improved clarity and tracking.


      Similarly, when the user applies a filter, such as selecting "Resolved" in the Status filter, only tickets with the "Resolved" status will be displayed. The total count of these filtered tickets will be shown for better visibility and tracking.
