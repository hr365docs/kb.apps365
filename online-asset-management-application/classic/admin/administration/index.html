Custom Columns


    Here you can define custom fields for both Assets and Users.

    • For Assets: Enter the name in the Asset Property Name field, select the Property Column Type, and then choose the Column Category as Asset.

      No need to select the SPO user properties field.


      This custom field you can see while adding asset as shown below.


      To add the created custom field column for assets in the table, go to Column Settings, select the checkbox of the created custom field, and click on the Submit button.


    Display control on custom field.

    By default, the display status of a custom field is enabled.

    • You can control the display of a custom field from the Column Settings page and in the User table. If the status of a custom field is disabled for the column category type Asset, that custom field will not be visible on the Column Settings (Column Order) page.

    • If the status of a custom field is disabled for the column category type User, the custom field will not be visible on the Add User popup page or in the User table.


  • Delete - Select the check box of custom field and click on Delete button.

  • Edit - You can edit only custom field name, just select check box of custom field and click on Edit button, edit popup page will display change name in asset property name field in order to change custom field name and click on Submit button.

  • Add Asset


      1. New Assets – Admins can add new assets from this section. To add assets, follow the steps in Add Asset.

      2. Edit Assets – Click the Edit Asset tab to open a popup edit page. Use the edit icon to enter edit mode, make changes, and click the Update button. Alternatively, click the asset's hyperlink from the Home or Assign Assets table to open the edit popup.


      3. Delete Assets – Select the desired asset(s) and click the Delete Asset tab to remove them. You can delete up to 25 assets at once.

      4. Display Assets – View asset details, edit information, add documents, or print QR codes from this section.

      5. Clone Asset – To create a new asset similar to an existing one, select the asset and click the Clone Asset tab. Edit the data as needed or keep it unchanged, then click Submit.


      6. Export Barcodes – Use this option to export asset barcodes.

      7. Export QR Codes – Use this option to export asset QR codes.

  • Add Status

      From here you can able to add status.


      To add a new asset status, follow the steps below:

      1. Click on the +Add button.

      2. Enter the desired Asset Status and select the checkboxes as per your requirements.

      3. Click on the Save button to save the status.

      4. To add multiple statuses, click on the Save and Add More button.

      5. The added Status will appear in the dropdown on the + Asset or + New Asset page.



      Edit: Click the Edit Asset status button, this display Edit Asset Type page. Update the entries and click on Update button.

      Delete: Click the delete button, this displays a dialog box. Click on Yes button.

  • Search Index

      From here, admin can add or remove the search index. The Added search index will display on the home page. Admin/users can find the details from the search index as per added column name.

      The maximum allowed search column entries are 18.


      To add the search index, follow the below steps:


      1. Click on Search Index under administartion. This displays Search Index page.

      2. Click on +Add and enter Search Column.

      3. Click on the Submit button.


      To remove, select the Action and click on the Remove button.

  • Acknowledgement Status

      On this page, those assets displayed which are not yet email notified to users about assigned. There we can find assets.

      • When assets are assigned from bulk asset using CSV file,
      • If we assign assets when asset accept notification is disabled in general settings.
      • Synced Intune assets.

      To send assigned asset notification to users select the assets from action column and click on initiate now.

      Allow to send notification for 25 assets at a time.

  • Equipments

      From here, admin can add or edit or delete equipment.

      To add equipment, follow the below steps:

      Add

      1. Click on the +Add Equipment, this displays Add Equipment page.

      2. Enter the Unique Id, Item Name,Purchase Date, Quantity, Status and Item Description.


      3. Click on the Save button.

      Bulk Add

      1. Click on +Bulk Equipments, this displays Add Multiple Equipments.

      2. Choose the data loaded CSV file from system and upload the file.


      3. Click on the Upload button.

      To edit, select the Action and click on the Edit button. Update the entries and click on Submit button.


      To delete, select the Action and click on the Delete button.
