Home page

On the homepage, you will find various tabs such as "Status", "My Tasks", "Weekly Timesheets", "Approval Task Status", and "Log Weekly Timesheets" as shown in the image below.


  1. Weekly Timesheets

    1. Setup

      2. Admins can customize weekly timesheets based on their requirements before use. Below are the key features for customization.

      By default, all features are disabled. You can enable and configure them based on your needs.

      You can click on the "Save Settings" button after making changes in each tab on the settings page.

      Time Tracking Days: Block weekends by going to Settings > Advanced Settings > Time Tracking Days. Uncheck the weekend days and save the settings.

      Restrict Daily and Weekly Hours: Configure max daily and weekly hours by enabling the respective toggle in Settings under Features tab, set the limits, and save the settings.

      Activities: Enabling this will show the activity column in the weekly timesheet. To enable it, go to Settings > General, toggle "Display Activities" on, and save the settings.

      Work Place: This feature displays the work place column, where users can select their work location. This is particularly important for billing purposes. To enable it, go to Settings > Features and toggle "Work Place" on. You can add more work places and save the settings.

      Do not edit or remove the "On-site" option in the work place popup, as it is essential for accurate onsite billing.

      Mandatory Task Selection: By default, task selection is mandatory when tracking time. To disable it, go to Settings > Features, toggle off "Mandatory task selection while tracking time," and save the settings.

    2. Submitting Weekly Timesheets

      3. Submit timesheets for the past week or on the last working day of the current week. You can do this from the "Weekly Timesheets" tab or the side panel under "Timesheets." To submit:

      • Select the week number.
      • Choose the project and corresponding task from the dropdown menu.
      • Enter time and comments, and then click "Submit" to save.

    3. Submitting Weekly Timesheets for Multiple Projects or Tasks

      4. You can submit timesheets for multiple projects or tasks by clicking the "Add" button to include them in your weekly timesheet.

    4. Saving Timesheets

      5. Save timesheets for later submission by clicking the "Save" button. These will appear in drafts, which can be viewed by refreshing the week calendar.

    5. Weekly Timesheet Cancellation

      6. Users can cancel their submitted timesheets as long as they are not approved. To cancel, click the "Cross" icon located on the right side of the weekly timesheet table.

  2. Track Time or Daily Timesheets

    3. The "Track Time" tab is disabled by default. To enable it, go to Settings > Features, toggle on "Daily Time Tracking," and the track time option will be available as shown below.

    1. Setup

      2. Some settings from the Weekly Timesheet are also applicable to Track Time. Admins can set up Track Time based on their requirements before enabling it.

      Tracker Column Setup: By default, three tracking columns are displayed: "Track Time," "Log Time," and "Duration." To disable any of them, go to Settings > Features and uncheck the respective option in the "Daily Time Tracking Options" row, then save the settings.

      Enable Daily Timesheet Report Notification: This option is enabled by default. You can disable it under "Daily Time Tracking Options" in the Settings.

      Max Limit of Hours per Day: To set the maximum daily time limit for "Track Time," enable "Restrict Daily and Weekly Hours" in the settings, define the limit, and save the settings.

    2. Submitting Daily Timesheets

      3. To submit a daily timesheet, select the date, project, task, and enter the time and memo. Then click "Save." After saving, select the checkbox and click "Submit" to finalize the entry.

      If you submit a daily timesheet, it will also reflect in the weekly timesheet table where you can cancel it if needed.

      You can submit timesheets for multiple projects or tasks by repeating the process and selecting multiple checkboxes before submitting.

      Note: Drafts for daily timesheets are not supported.

  3. Status

    4. The "Status" tab provides the status of your submitted timesheets.

  4. My Tasks

    5. Under the "My Tasks" tab, users can find tasks assigned to them.

  5. Approval

    6. Project managers can approve timesheets from the "Approval" tab. To approve, select the week, employee, and the timesheet entry. Then, use the "Approve" or "Reject" button to finalize the decision.

    Program managers can approve project managers' timesheets.

  6. Log Weekly Timesheet As

    7. This feature allows admins and program managers to submit timesheets on behalf of users. By default, it is disabled. To enable it, go to Settings > Advanced Settings, and toggle on "Time Tracking Delegation by Admin/PM."

    To log a timesheet, go to the "Weekly Timesheets" tab, select the user and week number, fill in the details, and click "Submit."

    Follow the same steps to submit daily timesheets on behalf of users.

  7. Task Status

    8. Project managers can view the task status of employees from the "Task Status" tab and notify users if their task deadline is overdue.

    You can filter the task status and export the filtered data to an Excel file by clicking the "Export" link at the top-right of the table.

    To notify users, select the items in the action column, click "Notify," add comments in the popup, and click "Submit" to send the notification via email.

  8. Self-Assign Task (or Home Page Task Tab)

    9. Users can create their own tasks from the "Task" tab, which is disabled by default. Admins can enable it from Settings > Features by toggling on the "Self-Assign Task" option.

    Allow Self-Assign Task Billable: By default, billable tasks are disabled. Admins can enable this option from Settings > Features by toggling on the "Allow Self-Assign Task Billable" option.
